## 🐱🛡️ API Defender: Hack & Save the System

# A fun little cybersecurity game built by TIGDAM — three curious minds creating our very first project together.

# API Defender lets you play as both the hacker 😈 and the defender 🛡️, breaking intentionally vulnerable APIs and then fixing them to learn how real-world API security works.

# Think of it as:

## a mini hacking game + learning playground + chaos (but educational).
## 🎮 Features of Pure Controlled Chaos

# 💥 Exploit vulnerable APIs
# 🐱 Cause dramatic fake data leaks
# 🛡️ Defend systems like a security wizard
# 🏆 Collect badges & XP
# 📜 Watch live request/response chaos unfold
# 🤖 Get hints from our slightly suspicious system bot

# ⚙️ Built With
# Frontend: HTML, CSS, JavaScript
# Backend: Python + FastAPI
# Team: TIGDAM 💥

## 🚀 Why We Built This


# As our first project, we wanted to create something that was both fun and educational.
# Instead of just explaining API security, we built a game where users can actually interact with vulnerabilities and understand how they can be fixed.

# Because learning security should feel like a game, not a headache.

# Everything here is intentionally vulnerable for learning purposes.
